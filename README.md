# 山菜図鑑アプリ

山菜の写真、詳細情報、採取記録、写真ストック、擬人化図鑑をまとめた静的Webアプリです。

## 使い方

`index.html` をブラウザで開くと起動します。記録や写真ストック、擬人化少女の解放状態は、ブラウザ内に保存されます。

## GitHub Pagesで公開する場合

1. このフォルダの中身をGitHubリポジトリにアップロードします。
2. GitHubの `Settings` から `Pages` を開きます。
3. `Deploy from a branch` を選び、`main` ブランチの `/root` を指定します。
4. 表示されたURLからブラウザでアプリを開けます。

## スマホでアプリのように使う場合

GitHub Pagesの公開URLをスマホのブラウザで開き、ブラウザの共有メニューから「ホーム画面に追加」を選ぶと、アプリのように起動できます。
